# FoxTouch

**English** | [简体中文](README_zh-CN.md) | [繁體中文](README_zh-TW.md) | [日本語](README_ja.md) | [한국어](README_ko.md) | [Español](README_es.md) | [Bahasa Melayu](README_ms.md)

An AI-powered phone agent for Android, inspired by [Claude Code](https://docs.anthropic.com/en/docs/claude-code). FoxTouch observes the screen via Accessibility Services, understands UI elements, and performs actions on behalf of the user through natural language instructions — bringing the agentic coding experience to the phone screen.

### How It Works

1. User describes a task in natural language
2. FoxTouch reads the screen using the Android Accessibility API
3. An LLM (configurable) analyzes the UI and decides actions
4. FoxTouch executes actions (tap, type, scroll, swipe, etc.) via Accessibility Services
5. The observe-think-act cycle repeats until the task is complete

### Use Cases

**Daily Automation**

> **You:** "Help me order food delivery — I want to spend around ¥30, pick dishes based on the taste preferences I told you yesterday, and use the best coupon to make it as cheap as possible"
>
> FoxTouch recalls your preferences from memory → opens Meituan → browses restaurants → plans an order matching your budget and taste → applies the best coupon → asks for your confirmation before paying

> **You:** "Send a WeChat message to Mom saying I'll be home late, and set an alarm for 7am tomorrow"
>
> FoxTouch opens WeChat → finds Mom → types and sends the message → opens Clock → creates the alarm → confirms both tasks done

**Information & Research**

> **You:** "Compare the price of AirPods Pro on JD and Taobao, and tell me which is cheaper"
>
> FoxTouch opens JD → searches and notes the price → opens Taobao → searches and notes the price → reports the comparison with links

> **You:** "Translate everything on this screen to English"
>
> FoxTouch reads the current screen content → provides a full translation in the chat

**Social Media**

> **You:** "Post my latest photo to WeChat Moments with a caption about the sunset"
>
> FoxTouch opens WeChat Moments → starts a new post → selects the latest photo → writes a caption → asks for your approval before posting

> **You:** "Check what's trending on Weibo and give me a summary"
>
> FoxTouch opens Weibo → browses the trending list → scrolls through top topics → summarizes the key stories in the chat

**Routine Tasks & Gaming**

> **You:** "Do my daily check-in on Genshin Impact"
>
> FoxTouch opens the game → navigates to the check-in page → claims the daily reward → saves this as a reusable skill for next time

> **You:** "Go through all my unread notifications and tell me if anything is urgent"
>
> FoxTouch opens the notification shade → reads each notification → filters and summarizes the important ones → dismisses the rest

### Key Features

- **Multi-provider LLM support** — OpenAI, Anthropic Claude, Google Gemini, OpenRouter, and any OpenAI-compatible API
- **Screen understanding** — UI element tree parsing + screenshot annotation with coordinate grids and element labels
- **Full device interaction** — tap, type, scroll, swipe, long press, pinch, back/home navigation, app launch
- **Plan mode** — complex tasks get planned and reviewed before execution
- **Skill system** — save and reuse plans for recurring workflows
- **Task tracking** — multi-step progress visible to the user in real-time
- **Floating overlay** — control the agent from any app via a bottom bar overlay
- **Built-in IME** — invisible input method for reliable text input and clipboard access across all apps
- **Enhanced screenshot** — MediaProjection fallback for apps that block accessibility screenshots (e.g., WeChat)
- **Context compaction** — automatic conversation summarization to stay within token limits
- **Voice input & TTS** — speak to the agent and hear responses read aloud
- **Persistent memory** — the agent remembers instructions and context across sessions
- **Safety guardrails** — per-tool approval, risk-level permissions, YOLO mode toggle
- **Multi-language UI** — English, 简体中文, 繁體中文, 日本語, 한국어, Español, Bahasa Melayu

### Requirements

- Android 11+ (API 30)
- Accessibility Service permission
- Overlay (draw over other apps) permission
- An API key for at least one supported LLM provider

### Build

```bash
git clone https://github.com/Foxerine/FoxTouch.git
cd FoxTouch
./gradlew assembleDebug
```

### Architecture

- **Language**: Kotlin 2.1 with Jetpack Compose
- **UI Framework**: Material 3
- **DI**: Hilt
- **Database**: Room (sessions, messages, tasks)
- **Networking**: Ktor Client + OpenAI Kotlin SDK
- **Encryption**: Google Tink for API key storage
- **State Machine**: Tinder StateMachine for agent lifecycle
