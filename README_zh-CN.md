# FoxTouch

[English](README.md) | **简体中文** | [繁體中文](README_zh-TW.md) | [日本語](README_ja.md) | [한국어](README_ko.md) | [Español](README_es.md) | [Bahasa Melayu](README_ms.md)

FoxTouch 是一个 AI 驱动的 Android 手机智能体，灵感来自 [Claude Code](https://docs.anthropic.com/en/docs/claude-code)。它通过无障碍服务观察屏幕、理解 UI 元素，并根据自然语言指令代替用户执行操作 — 将 AI 编程助手的智能体体验带到手机屏幕上。

> **状态：概念验证 / 开发中**
>
> 本项目处于早期实验阶段。功能可能不完整、不稳定，或发生重大变更。

### ⚠️ 重要声明

**AI 生成代码**：本项目绝大部分代码由 AI（Claude）生成，由人类进行方向指导和审查。包括应用架构、UI 实现、工具系统设计、无障碍集成和系统提示词。

**使用风险自负**：本软件按"原样"提供，不作任何保证。AI 智能体可以在你的手机上执行点击、输入、滑动、启动应用等操作。**你需要对使用本软件产生的任何后果承担全部责任。** 作者不对因使用本软件可能导致的任何损害、数据丢失、意外购买或其他问题承担责任。

### 工作原理

1. 用户用自然语言描述任务
2. FoxTouch 通过 Android 无障碍 API 读取屏幕
3. LLM（可配置）分析 UI 并决定操作
4. FoxTouch 通过无障碍服务执行操作（点击、输入、滚动、滑动等）
5. "观察-思考-执行"循环重复，直到任务完成

### 使用场景

**日常自动化**

> **你：** "帮我点个外卖，大概花 30 块，按我昨天告诉你的口味偏好来选，用最划算的优惠券凑到最便宜"
>
> FoxTouch 从记忆中回忆你的口味偏好 → 打开美团 → 浏览餐厅 → 规划符合预算和口味的订单 → 应用最优优惠券 → 付款前请你确认

> **你：** "给妈妈发微信说我晚点到家，顺便设个明早 7 点的闹钟"
>
> FoxTouch 打开微信 → 找到妈妈 → 输入并发送消息 → 打开时钟 → 创建闹钟 → 确认两个任务已完成

**信息查询与整理**

> **你：** "帮我对比京东和淘宝上 AirPods Pro 的价格，告诉我哪个便宜"
>
> FoxTouch 打开京东 → 搜索并记录价格 → 打开淘宝 → 搜索并记录价格 → 汇报对比结果

> **你：** "把屏幕上的内容翻译成英文"
>
> FoxTouch 读取当前屏幕内容 → 在对话中提供完整翻译

**社交媒体**

> **你：** "把我最新的照片发到朋友圈，配文写个关于日落的"
>
> FoxTouch 打开微信朋友圈 → 新建动态 → 选择最新照片 → 撰写配文 → 发布前请你确认

> **你：** "看看微博热搜都有什么，给我总结一下"
>
> FoxTouch 打开微博 → 浏览热搜榜 → 滚动查看热门话题 → 在对话中总结要点

**日常签到与通知**

> **你：** "帮我去原神签到"
>
> FoxTouch 打开游戏 → 导航到签到页 → 领取每日奖励 → 保存为可复用技能，下次直接执行

> **你：** "看看我所有未读通知，有没有什么重要的"
>
> FoxTouch 打开通知栏 → 逐条阅读通知 → 筛选并总结重要信息 → 清除其余通知

### 核心功能

- **多模型支持** — OpenAI、Anthropic Claude、Google Gemini、OpenRouter 及任何 OpenAI 兼容 API
- **屏幕理解** — UI 元素树解析 + 带坐标网格和元素标注的截图
- **全面的设备交互** — 点击、输入、滚动、滑动、长按、捏合、返回/主页导航、启动应用
- **计划模式** — 复杂任务先规划再执行，用户审核后才运行
- **技能系统** — 保存并复用操作方案
- **任务跟踪** — 实时展示多步骤执行进度
- **悬浮窗** — 通过底部悬浮栏在任意应用中控制智能体
- **内置输入法** — 隐形输入法，实现跨应用的可靠文本输入和剪贴板访问
- **增强截图** — 对阻止无障碍截图的应用（如微信）使用 MediaProjection 备选方案
- **上下文压缩** — 自动摘要对话以保持在 token 限制内
- **语音输入与朗读** — 语音对话 + 朗读回复
- **持久记忆** — 智能体跨会话记住指令和上下文
- **安全防护** — 逐工具审批、风险等级权限、YOLO 模式切换
- **多语言界面** — English、简体中文、繁體中文、日本語、한국어、Español、Bahasa Melayu
