# FoxTouch

[English](README.md) | [简体中文](README_zh-CN.md) | [繁體中文](README_zh-TW.md) | [日本語](README_ja.md) | **한국어** | [Español](README_es.md) | [Bahasa Melayu](README_ms.md)

FoxTouch는 [Claude Code](https://docs.anthropic.com/en/docs/claude-code)에서 영감을 받은 AI 기반 Android 스마트폰 에이전트입니다. 접근성 서비스를 통해 화면을 관찰하고, UI 요소를 이해하며, 자연어 지시에 따라 사용자를 대신하여 작업을 수행합니다 — AI 코딩 어시스턴트의 에이전트 경험을 스마트폰 화면으로.

> **상태: 개념 증명 / 개발 중**
>
> 이 프로젝트는 초기 실험 단계에 있습니다. 기능이 불완전하거나, 불안정하거나, 크게 변경될 수 있습니다.

### ⚠️ 중요 고지사항

**AI 생성 코드**: 이 프로젝트의 코드 대부분은 AI(Claude)에 의해 생성되었으며, 인간이 방향 지시와 검토를 수행했습니다. 애플리케이션 아키텍처, UI 구현, 도구 시스템 설계, 접근성 통합 및 시스템 프롬프트를 포함합니다.

**사용에 따른 위험은 본인 부담**: 이 소프트웨어는 어떠한 보증도 없이 "있는 그대로" 제공됩니다. AI 에이전트는 탭, 입력, 스와이프, 앱 실행 등 휴대폰에서 작업을 수행할 수 있습니다. **이 소프트웨어 사용으로 인한 모든 결과에 대해 사용자가 전적으로 책임집니다.** 저작자는 사용으로 인해 발생할 수 있는 손해, 데이터 손실, 의도하지 않은 구매 또는 기타 문제에 대해 어떠한 책임도 지지 않습니다.

### 작동 방식

1. 사용자가 자연어로 작업을 설명
2. FoxTouch가 Android 접근성 API로 화면을 읽음
3. LLM(설정 가능)이 UI를 분석하고 작업을 결정
4. FoxTouch가 접근성 서비스로 작업을 수행 (탭, 입력, 스크롤, 스와이프 등)
5. 작업이 완료될 때까지 "관찰-생각-실행" 사이클 반복

### 사용 사례

**일상 자동화**

> **당신:** "배달 음식 주문해줘, 3만원 정도로, 어제 알려준 내 취향대로 골라서, 가장 저렴한 쿠폰 써줘"
>
> FoxTouch가 메모리에서 취향을 불러옴 → 배달 앱 열기 → 음식점 탐색 → 예산과 취향에 맞는 주문 계획 → 최적 쿠폰 적용 → 결제 전 확인 요청

> **당신:** "엄마한테 카카오톡으로 늦게 간다고 보내고, 내일 아침 7시 알람 설정해줘"
>
> FoxTouch가 카카오톡 열기 → 엄마 찾기 → 메시지 입력 후 전송 → 시계 앱 열기 → 알람 생성 → 두 작업 완료 확인

**정보 검색 및 정리**

> **당신:** "쿠팡이랑 네이버에서 AirPods Pro 가격 비교해서 어디가 더 싼지 알려줘"
>
> FoxTouch가 쿠팡 열기 → 검색 후 가격 기록 → 네이버 열기 → 검색 후 가격 기록 → 비교 결과 보고

> **당신:** "이 화면 내용을 영어로 번역해줘"
>
> FoxTouch가 현재 화면 내용 읽기 → 채팅에서 전체 번역 제공

**소셜 미디어**

> **당신:** "최신 사진으로 인스타 스토리 올려줘, 노을 관련 캡션 달아서"
>
> FoxTouch가 Instagram 열기 → 스토리 생성 → 최신 사진 선택 → 캡션 작성 → 게시 전 확인 요청

**데일리 체크인 & 알림**

> **당신:** "원신 출석 체크 해줘"
>
> FoxTouch가 게임 열기 → 출석 페이지 이동 → 일일 보상 수령 → 다음에 재사용할 수 있는 스킬로 저장

> **당신:** "읽지 않은 알림 전부 확인해서 중요한 거 있으면 알려줘"
>
> FoxTouch가 알림 패널 열기 → 각 알림 읽기 → 중요한 항목 필터링 및 요약 → 나머지 정리

### 주요 기능

- **다중 LLM 지원** — OpenAI, Anthropic Claude, Google Gemini, OpenRouter 및 모든 OpenAI 호환 API
- **화면 이해** — UI 요소 트리 파싱 + 좌표 그리드와 요소 레이블이 포함된 스크린샷 주석
- **전체 기기 조작** — 탭, 입력, 스크롤, 스와이프, 길게 누르기, 핀치, 뒤로/홈, 앱 실행
- **계획 모드** — 복잡한 작업을 사전 계획 후 사용자 승인 후 실행
- **스킬 시스템** — 액션 플랜 저장 및 재사용
- **작업 추적** — 다단계 진행 상황 실시간 표시
- **플로팅 오버레이** — 모든 앱에서 에이전트 제어
- **내장 입력기** — 모든 앱에서 안정적인 텍스트 입력과 클립보드 접근을 위한 보이지 않는 입력 방식
- **향상된 스크린샷** — 접근성 스크린샷을 차단하는 앱을 위한 MediaProjection 폴백
- **컨텍스트 압축** — 토큰 한도 내 유지를 위한 대화 자동 요약
- **음성 입력 및 TTS** — 음성 대화 + 응답 읽어주기
- **영구 메모리** — 세션 간 지시사항과 컨텍스트 기억
- **안전 장치** — 도구별 승인, 위험 수준 권한, YOLO 모드 토글
- **다국어 UI** — English, 简体中文, 繁體中文, 日本語, 한국어, Español, Bahasa Melayu

### 다운로드

Release APK는 약 **9 MB**에 불과합니다.

### 요구 사항

- Android 11+ (API 30)

- 오버레이 (다른 앱 위에 표시) 권한
