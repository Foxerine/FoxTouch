# FoxTouch

[English](README.md) | [简体中文](README_zh-CN.md) | **繁體中文** | [日本語](README_ja.md) | [한국어](README_ko.md) | [Español](README_es.md) | [Bahasa Melayu](README_ms.md)

FoxTouch 是一個 AI 驅動的 Android 手機智慧體，靈感來自 [Claude Code](https://docs.anthropic.com/en/docs/claude-code)。它透過無障礙服務觀察螢幕、理解 UI 元素，並根據自然語言指令代替使用者執行操作 — 將 AI 程式助手的智慧體體驗帶到手機螢幕上。

> **狀態：概念驗證 / 開發中**
>
> 本專案處於早期實驗階段。功能可能不完整、不穩定，或發生重大變更。

### ⚠️ 重要聲明

**AI 生成程式碼**：本專案絕大部分程式碼由 AI（Claude）生成，由人類進行方向指導和審查。包括應用架構、UI 實作、工具系統設計、無障礙整合和系統提示詞。

**使用風險自負**：本軟體按「原樣」提供，不作任何保證。AI 智慧體可以在你的手機上執行點擊、輸入、滑動、啟動應用程式等操作。**你需要對使用本軟體產生的任何後果承擔全部責任。** 作者不對因使用本軟體可能導致的任何損害、資料遺失、意外購買或其他問題承擔責任。

### 運作原理

1. 使用者用自然語言描述任務
2. FoxTouch 透過 Android 無障礙 API 讀取螢幕
3. LLM（可設定）分析 UI 並決定操作
4. FoxTouch 透過無障礙服務執行操作（點擊、輸入、捲動、滑動等）
5. 「觀察-思考-執行」循環重複，直到任務完成

### 使用場景

**日常自動化**

> **你：** 「幫我點個外賣，大概花 30 塊，按我昨天告訴你的口味偏好來選，用最划算的優惠券湊到最便宜」
>
> FoxTouch 從記憶中回憶你的口味偏好 → 開啟外送平台 → 瀏覽餐廳 → 規劃符合預算和口味的訂單 → 套用最優優惠券 → 付款前請你確認

> **你：** 「給媽媽傳 LINE 說我晚點到家，順便設個明早 7 點的鬧鐘」
>
> FoxTouch 開啟 LINE → 找到媽媽 → 輸入並傳送訊息 → 開啟時鐘 → 建立鬧鐘 → 確認兩個任務已完成

**資訊查詢與整理**

> **你：** 「幫我對比蝦皮和 momo 上 AirPods Pro 的價格，告訴我哪個便宜」
>
> FoxTouch 開啟蝦皮 → 搜尋並記錄價格 → 開啟 momo → 搜尋並記錄價格 → 匯報對比結果

> **你：** 「把螢幕上的內容翻譯成英文」
>
> FoxTouch 讀取當前螢幕內容 → 在對話中提供完整翻譯

**社群媒體**

> **你：** 「幫我發一則 IG 限時動態，用最新的照片配個日落的文字」
>
> FoxTouch 開啟 Instagram → 建立限時動態 → 選擇最新照片 → 撰寫文字 → 發布前請你確認

> **你：** 「看看 PTT 熱門文章都有什麼，給我總結一下」
>
> FoxTouch 開啟 PTT → 瀏覽熱門看板 → 捲動查看熱門文章 → 在對話中總結要點

**日常簽到與通知**

> **你：** 「幫我去原神簽到」
>
> FoxTouch 開啟遊戲 → 導航到簽到頁 → 領取每日獎勵 → 儲存為可複用技能，下次直接執行

> **你：** 「看看我所有未讀通知，有沒有什麼重要的」
>
> FoxTouch 開啟通知欄 → 逐條閱讀通知 → 篩選並總結重要資訊 → 清除其餘通知

### 核心功能

- **多模型支援** — OpenAI、Anthropic Claude、Google Gemini、OpenRouter 及任何 OpenAI 相容 API
- **螢幕理解** — UI 元素樹解析 + 帶座標網格和元素標註的截圖
- **全面的裝置互動** — 點擊、輸入、捲動、滑動、長按、捏合、返回/主畫面導航、啟動應用程式
- **計畫模式** — 複雜任務先規劃再執行，使用者審核後才運行
- **技能系統** — 儲存並複用操作方案
- **任務追蹤** — 即時展示多步驟執行進度
- **懸浮視窗** — 透過底部懸浮列在任意應用程式中控制智慧體
- **內建輸入法** — 隱形輸入法，實現跨應用程式的可靠文字輸入和剪貼簿存取
- **增強截圖** — 對阻止無障礙截圖的應用程式（如微信）使用 MediaProjection 備選方案
- **上下文壓縮** — 自動摘要對話以保持在 token 限制內
- **語音輸入與朗讀** — 語音對話 + 朗讀回覆
- **持久記憶** — 智慧體跨工作階段記住指令和上下文
- **安全防護** — 逐工具審批、風險等級權限、YOLO 模式切換
- **多語言介面** — English、简体中文、繁體中文、日本語、한국어、Español、Bahasa Melayu
